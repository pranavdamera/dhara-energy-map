# Data Licenses (Week 1)

This project uses public datasets for research/portfolio demonstration in one district (Anantapur).

- geoBoundaries (IND ADM2): open administrative boundaries, check source page for exact terms.
- OpenStreetMap (roads, substations): [ODbL](https://www.openstreetmap.org/copyright)
- Sentinel-2 SR Harmonized (`COPERNICUS/S2_SR_HARMONIZED`): see Earth Engine dataset terms.
- ESA WorldCover v200 (`ESA/WorldCover/v200`): [CC BY 4.0](https://creativecommons.org/licenses/by/4.0/)
- USGS SRTMGL1_003: public dataset in Earth Engine catalog, see provider terms.
- Global Solar Atlas: free public GIS downloads, cite source and usage terms in reports.

Always include source attribution in downstream outputs and reports.
